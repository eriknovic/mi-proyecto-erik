\begin{frame}
\frametitle{Integración de los Recursos de Información}
\justifying
La \textbf{\textit{integración de los recursos}} es el proceso de búsqueda y recuperación significativa de información existente en los recursos, para responder una consulta dada por un usuario.

Las actividades básicas para la integración de los recursos son:
\begin{itemize}
\item Representar el conocimiento de los recursos en un modelo.
\item Consultar información en el modelo.
\end{itemize}

\end{frame}

\section{Integración semántica de recursos}
\begin{frame}
\frametitle{Integración semántica de recursos}
\justifying
El uso de las tecnologías semánticas para realizar el proceso de integración de información para los recursos de una memoria corporativa.

Las etapas de esta integración semántica son:
 
\begin{enumerate}
\item \justifying representar el conocimiento explícito de los recursos en un formato estándar.
\item \justifying enriquecer el conocimiento implícito mediante la introducción de axiomas.
\item \justifying buscar y recuperar la información en este modelo para responder una pregunta dada.
\end{enumerate}
\end{frame}
